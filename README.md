## Real-Time Object Detection with Measurement Using OpenCV

A Python project that performs real-time object detection and measures object dimensions using a webcam. The program detects multiple objects, draws bounding boxes, labels them, and estimates their size based on pixel measurements.

Optional enhancements: Save detected frames, video, or integrate with IoT devices
